# 简介
基于springboot+mybatisplus+vue+element-ui的博客和博客管理系统

该项目分为博客和博客管理系统两部分

管理员账号admin    密码112233

# 技术栈

核心框架：springboot

持久层框架：mybatisplus

数据库连接池：druid 

接口测试框架：Swagger2

pojo框架：lombok

js框架：vue2.x

ui框架：element-ui

# 使用方法
1、创建数据库blog

2、执行doc/blog.sql

3、修改config/application.properties下的数据库和邮件配置信息

4、修改config/file.properties下的upload.path（图片上传存储目录）
